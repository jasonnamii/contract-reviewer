# Golden Eval Cases — Contract Consulting (5 cases)

## E1. SHA (주주간계약) — 시리즈 B
### Input
- 시리즈 B 50억 투자, VC 지분 15%, 창업자 65%
- VC가 제시한 SHA 초안
### Golden 핵심 점검
1. **Drag-Along threshold** — 67%↑ (50% 거절)
2. **Anti-dilution** — Weighted Average (Broad-based) (Full-ratchet 거절)
3. **ROFR + Co-sale** — VC만이 아닌 양방향
4. **Board** — 창업자 3 / VC 1 / 독립 1
5. **Protective Provisions** — 합리적 범위 (M&A·신주발행·차입 등 핵심)
6. **Liquidation Preference** — 1x non-participating (1x participating 거절)
7. **자사주 활용 금지** — 2026.1 소각의무 반영
8. **준거법·관할** — 한국법·서울중앙

## E2. NDA (상호) — M&A 실사
### Input
- M&A 검토 단계, 양측 기밀 교환
### Golden 핵심
1. **상호적용** (일방형 ✗)
2. **기간** — 2년 (장기 5년 → 영업비밀법 별도)
3. **Permitted Disclosure** — 변호사·회계사·자문사 (NDA 부담)
4. **No solicitation** — 핵심인력 1년 (의제 협상 카드)
5. **Carve-out** — 공지·독립개발·합법취득
6. **Return/Destroy** + Backup 데이터 면제
7. **준거법 + 분쟁** — 한국법·서울중재

## E3. SPA (인수계약) — 비상장 인수
### Input
- 비상장사 100% 인수, 거래대금 500억
### Golden 핵심
1. **진술보장** — Fundamental + General + Tax + Environmental 4층
2. **Knowledge qualifier** — 임원 5명+ Constructive Knowledge
3. **Cap** — General 30% / Fundamental 100% / Tax-Env 별도
4. **Basket** — Tipping 1% (Deductible 거절)
5. **Survival** — General 24개월 / Fundamental 무한 / Tax 시효
6. **MAE carve-out** — Industry-wide + 12개월 + disproportionate
7. **Earn-out** — EBITDA 산정 명시 + 매수인 사업운영 자율 (서울고법 2025나2034567)
8. **Closing 조건** — Regulatory + Material Consents
9. **준거법** — 한국법·서울중재 (KCAB)

## E4. SaaS MSA — B2B SaaS
### Input
- 미국 SaaS 도입, 연 5억, 데이터 한국 처리
### Golden 핵심
1. **SLA** — 99.9% uptime + Credit
2. **DPA 부속** — 개보법 §28의8 + GDPR
3. **국외이전** — SCC 또는 적정성 결정 (개보법)
4. **AI 학습 옵트아웃** — 입력 데이터 학습 ✗ (2026.1 AI 기본법)
5. **IP** — 사용자 데이터 100% 사용자, 모델은 SaaS
6. **Termination** — Convenience 90일 + 데이터 반환 60일
7. **준거법** — Delaware → 한국법 카운터 (실패 시 SIAC 중재)
8. **Audit Right** — 연 1회

## E5. 하도급 계약 — 제조
### Input
- 대기업 → 중소 제조 하도급, 연 30억
### Golden 핵심
1. **부당특약 무효** (하도급법 §3의4, 2026.5)
2. **단가 결정** — 객관적 기준 + 변경 절차
3. **납기 변경** — 사전 협의 + 추가비용 정산
4. **기술자료 제공** — 정당한 사유·범위 한정·반환·파기
5. **하자담보** — 1년 (제조물책임 별도)
6. **노란봉투법** — 사용자성 분리 명시 (원청 단체교섭 차단)
7. **분쟁** — 하도급분쟁조정협의회 → 공정위 → 법원
8. **결제** — 60일 내 현금·전자어음 ✗
