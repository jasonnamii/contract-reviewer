# D3 상업거래 핵심 판례

→ 원본: VAULT/_skills research/contract-consulting/2026-05-07_judicial-corpus.md §D3

## 핵심 6건

### KCC-D3-001 거래상지위남용 (마이크로소프트)
- **사건:** 대법원 2003두5327 판결
- **룰:** 거래상지위 인정 → 일방적 조항 무효. 2025년 판례 "계속적 거래" 요건 폐지
- **계약 반영:** 거래상지위 의심 시 일방조항 자체 회피
- **신뢰도:** 상

### KCC-D3-002 하도급 부당특약
- **사건:** 공정위 하도급법 의결례 다수
- **룰:** 산업재해 비용 전가·일방적 단가 인하 = 시정명령 + 과징금 50-100% (2024-2025 가중)
- **신뢰도:** 상

### KCC-D3-003 대리점법 갱신 거절
- **사건:** 대리점법 시행 후 분쟁 (2016~)
- **룰:** 정당 사유 없는 갱신거절 = 손해배상. 대리점 신뢰이익 보호
- **신뢰도:** 상

### KCC-D3-004 가맹사업법 광고비 전가
- **사건:** 공정위 의결례
- **룰:** 가맹점주 동의 없는 광고비 전가 무효
- **신뢰도:** 상

### KCC-D3-005 MFN 일방 적용
- **사건:** 공정위 MFN 관련 의결
- **룰:** 거래상지위 + 일방 MFN = 거래상지위남용. 상호 MFN은 비교적 안전
- **신뢰도:** 중

### KCC-D3-006 일방적 변경권 무효
- **사건:** 약관규제법 §10 (일방적 변경권 금지)
- **룰:** 통지+동의·해지권 보장 없는 변경권 = 무효
- **신뢰도:** 상
