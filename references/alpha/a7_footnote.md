# α7 각주엔진 (Footnote Engine)

본문 인라인 각주 자동 합성. M2/M4 출력 포맷의 핵심.

## 각주 7요소

각 조항 검토 시 다음 7요소를 인라인 각주로 합성:

```
{조항 문구}
> [근거] {법조문 verbatim}
> [리스크] {중대/주의/통상} · 무효확률 ~{%} · 과징금 기대값 ~{금액}
> [처분례] {기관 사건번호} [신뢰도] (corpus_id)
> [판례] {법원 사건번호} [신뢰도] (case_id)
> [벤치마크] {시장 표준} (updated)
> [레버리지] {α5 레버 1+} → "{Tier2 1줄}"
> [가드레일] "{권장 수정안}"
```

## 검증 룰 (Hallucination 방어)

| 요소 | 검증 룰 |
|---|---|
| 법조문 | 정식명·조항번호 verbatim. "공정법" 약식 ✗ |
| 사건번호 | corpus_id 매칭. LLM 생성 ✗ |
| 처분례 | α1 카탈로그 inside만 |
| 판례 | α2 카탈로그 inside만 |
| 벤치마크 | α3 데이터 + 갱신일 |
| 무효확률 | "통계적 추정" 라벨 |
| 과징금 기대값 | α1 공식 + "통계적 추정" |

## 사용 예시

```
甲은 본 계약의 일체의 손해에 대하여 무한책임을 진다.
> [근거] 약관규제법 제7조(면책조항의 금지) 일부 적용
> [리스크] 중대 · 무효확률 ~75% · 과징금 기대값 N/A (사인 간)
> [판례] 대법원 2003두5327 [신뢰도: 상] (case_id: KCC-D3-001)
> [벤치마크] 한국 M&A Cap 거래대금 10-20% (updated: 2026-05-07)
> [레버리지] 시장표준 + 판례 → "Cap을 거래대금 15% + IP침해 무제한"
> [가드레일] "甲의 책임은 거래대금의 [15%]로 제한한다. 단, 고의·중과실, IP침해, 비밀유지의무 위반은 제외한다."
```

## 면제

M4 KAJ 카운터 모드에서 본문 자체가 카운터 redline 형식인 경우 각주는 별표(*)로 압축 가능.
