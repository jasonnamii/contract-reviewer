## 룰 추출 패턴

각 판례에서 다음 4요소 추출:
1. **사실관계 핵심** (계약 유형·당사자·쟁점 조항)
2. **법원 판단 근거** (적용 법조문·논리 골격)
3. **실무 룰** (협상에서 어떻게 활용?)
4. **계약서 반영** (문구 어떻게 쓸까?)

### 예시: 비경쟁 합리성 4요건 (대법원 2009다82244)

| 요건 | 판단 기준 |
|---|---|
| 보호이익 | 영업비밀·고객관계·투자비용 보호 정당성 |
| 기간 | 1-2년 합리적 (3년+ 무효 가능성 급증) |
| 지역 | 사용자 영업권 미치는 범위 (전국 광역 = 위험) |
| 대상 직무 | 동일·유사 직무 (전 산업 = 무효) |

→ 4요건 중 1개+ 결격 시 무효확률 50%+, 2개+ 결격 시 70%+

## 사용 예시

```
[판례] 대법원 2010.3.11. 선고 2009다82244 [신뢰도: 상]
(case_id: KCC-D4-002)
→ 룰: 비경쟁 4요건(보호이익·기간·지역·대상) 중 2개+ 결격 시 무효확률 70%+
→ 본 계약: 기간 3년·지역 전국·대상 동종 전부 → 3개 결격 → 무효확률 ~85%
```

## 사용 원칙

- 사건번호 verbatim. LLM 생성 ✗
- 검증 안된 사건은 "검증필요" 태깅 + 사용 금지
- 무효확률은 "통계적 추정" 라벨 강제 (절대규칙 11)
