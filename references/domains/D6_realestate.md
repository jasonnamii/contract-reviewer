# D6 — 부동산 계약

대표 계약: 주택·상가 임대차, 매매, 임대관리, 개발계약.

## 필수 조항

### 임대차
- **목적물** — 소재지·면적·용도
- **임대료·보증금** — 금액·지급일·연체이자
- **관리비** — 범위·계산·증감
- **임대기간** — 시작·종료·갱신
- **사용수익** — 용도·전대 금지
- **수선 의무** — 대수선 vs 소수선
- **원상복구** — 종료 시 회복의무·비용분담
- **해지 사유** — 차임연체·용도위반
- **Security Deposit 반환**

### 매매
- **목적물·대금·지급일정**
- **소유권 이전** — 등기 시기·비용
- **저당권·가압류 말소** 의무
- **하자담보** — 민법 580조
- **해약금** — 민법 565조
- **중개수수료·세금**

### 개발계약
- **토지매매·신탁·PF**
- **인허가 조건·시기**
- **공사진행·기성·준공**
- **지체상금·하자보증**

## Domain-Specific Redflag
- 과도한 원상복구 의무 (경미한 소모 포함)
- 임대료 임의 증액권 (편측)
- 전대·양도 절대 금지 (상가·대형 임대)
- 보증금 반환 조건 편측
- 계약갱신 거절 사유 추상적

## L5 Korea 연결
- **주택임대차보호법:** 대항력·우선변제·묵시갱신·계약갱신요구권(10년)
- **상가임대차보호법:** 환산보증금 기준·갱신요구권·권리금 보호
- **부동산거래신고법:** 신고의무·실거래가
- **K-A 민법:**
  - 제565 해약금
  - 제568 동시이행
  - 제580 매도인 하자담보
- **공인중개사법** — 중개보수 상한

## Tier 표준
- 원상복구: Tier1 자연소모 제외 / Tier2 기준일자 명시 / Tier3 현상태 반환
- 갱신: 주임법·상임법 강행규정 준수 필수 (편측 Opt-out 무효)
- 임대료 증액: Tier1 연 5% / Tier2 CPI 연동 / Tier3 협의 후 확정
