# D7 — 건설·공공조달 계약

대표 계약: FIDIC(Red/Yellow/Green Book), 시공, 설계·감리, 정부계약(나라장터), PF.

## 필수 조항

### 건설
- **공사범위·설계도서** — 명세·변경절차
- **공사기간** — 착공·준공·지체상금
- **공사대금** — 기성·Payment Schedule·유보금
- **변경·추가공사(Variation)** — 절차·단가
- **하자보증** — 기간(업종별 1-10년)·보증금
- **안전·환경** — 법령 준수
- **계약이행보증** — 계약보증금
- **준공검사·인수** — 절차

### FIDIC
- **Red Book** — 발주자 설계 시공
- **Yellow Book** — 시공자 설계·시공 (Design-Build)
- **Silver Book** — EPC 턴키
- **Clause 20 분쟁해결** — DAB→조정→중재

### 공공조달
- **국가계약법 / 지방계약법** 적용
- **일반조건·특수조건** — 조달청 표준
- **대금지급** — 14일 이내 원칙
- **계약보증금·하자보증금** — 10%·3%
- **지체상금** — 일 0.3%, 공사 0.0625%
- **부정당업자 제재** — 입찰참가 제한

## Domain-Specific Redflag
- 공기연장 청구권 부재 (Force Majeure·발주자 사유)
- Variation 단가 불명확 (신단가)
- 하자 범위 과도 (자연소모 포함)
- 지체상금 과다 (실손해 대비)
- 계약이행보증 몰수 조건 편측

## L5 Korea 연결
- **국가계약법 / 지방계약법 / 조달사업법**
- **건설산업기본법** — 등록·하도급 제한
- **K-E 하도급법** — 원도급자·하도급자 관계
- **건설기술진흥법** — 안전·품질
- **나라장터 G2B** — 전자조달
- **부정당업자 제재** — 입찰참가 제한 6개월~2년

## Tier 표준
- 지체상금: Tier1 법정률(공사 0.0625%) / Tier2 일 0.1% / Tier3 일 0.2% + 상한 10%
- Variation: Tier1 합의 단가 / Tier2 신단가 산정 / Tier3 제3자 감정
- 하자보증: 법정 업종별 기간 준수 필수 (단축 불가)

## 공공조달 특이
- 모든 조항이 국가계약법·특수조건 위배 불가
- 부당특약 = 무효 (국계법 제5조)
- 지체상금 감경 사유 명시 (천재지변·발주자 사유)
