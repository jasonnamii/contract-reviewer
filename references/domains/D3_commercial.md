# D3 — 상업거래 계약

대표 계약: 공급·판매·유통·라이선스·파트너십·대리점·가맹·프랜차이즈.

## 필수 조항
- **Scope of Services / Products** — 제공 범위·수량·품질
- **SLA (Service Level)** — 가동률·응답시간·Remedy
- **Pricing / Payment Terms** — 단가·할인·지급기한·지연이자
- **Acceptance** — 검수기준·기간·Deemed Acceptance
- **Warranty** — 기간·범위·Remedy
- **Exclusivity** — 독점 여부·기간·지역·제품 (주의: Yellow)
- **Territory** — 판매 지역
- **IP License** — 사용권 범위·Sublicense·Audit
- **Change of Control / Assignment** — 양도 가능 여부
- **Force Majeure** — 트리거·결과·기간
- **Insurance** — 유형·한도

## Domain-Specific Redflag
- Exclusivity 무제한 기간·지역
- MFN 폭넓음
- One-sided Termination for Convenience
- IP 포괄양도
- 자동갱신 조항 (Opt-out 어려움)

## L5 Korea 연결 (중요)
- **K-C 약관법:** 대량·반복 거래 시 약관 심사
- **K-D 공정거래법 제45조:** 거래상 지위 남용 (2024.12 심사지침)
- **K-E 3법 체크:**
  - 하도급법: 원사업자·수급사업자 관계면 서면교부 의무
  - 대리점법: 공급자·대리점 관계
  - 가맹사업법: 본부·가맹점 관계, 정보공개서 14일 전
- **표준계약서:** 공정위 업종별 표준계약서 준수도

## Tier 표준
- Exclusivity: Tier1 ✗ / Tier2 제한적 / Tier3 실적 연동 해제
- Termination: Tier1 For Cause 30d Cure / Tier2 Convenience 90d Notice / Tier3 180d+Fee
- IP: Tier1 Work Product만 / Tier2 Full + Background License / Tier3 범위 제한
