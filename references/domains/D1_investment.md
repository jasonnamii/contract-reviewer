# D1 — 투자금융 계약

대표 계약: SHA, SPA, Term Sheet, 전환사채(CB), RCPS, SAFE.

## 필수 조항
- **Preamble·Definitions** — 당사자·주식 종류
- **Share Subscription / Purchase** — 매수대금·배정·Closing
- **R&W (진술보장)** — 회사·주주 측 진술 (재무·소송·세무·IP)
- **Covenants** — 운영제한(Pre-closing), 경업금지·비경쟁(Post-closing)
- **Indemnification** — R&W 위반 시, Cap·Basket·Survival
- **Preferred Rights** — Liquidation Preference, Anti-dilution, Conversion
- **Protective Provisions** — 거부권 사항 (예산·신주·M&A)
- **Information Rights** — 재무보고·감사권
- **Tag-Along / Drag-Along** — 동반매각권·강제매각권
- **ROFR / ROFO** — 우선매수권
- **MFN** — 최혜국 조항 (주의: Yellow)
- **Exit / IPO** — 회수구조

## Domain-Specific Redflag
- Liquidation Preference 1x Non-Participating 초과 (2x+ = Red)
- Full Ratchet Anti-dilution (Weighted-Average 권장)
- 이사 과반수 지배 (소수주주 보호 부재)
- Drag-Along 낮은 비율 (50% 미만 = 위험)

## L5 Korea 연결
- K-G 2025.7.22 상법 382조의3 이사 충실의무 주주 확대
- K-G 2025.9.9 집중투표 의무화 (자산 2조원+)
- 자본시장법 공시의무 (상장사·5% 보고)

## Tier 표준
- Liquidation: Tier1 1x NP / Tier2 1x Participating Cap / Tier3 2x NP
- Anti-dilution: Tier1 Broad WA / Tier2 Narrow WA / Tier3 Full Ratchet
