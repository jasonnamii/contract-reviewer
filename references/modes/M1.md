# M1 — 계약 진단 (리뷰메모)

입력: 계약서 텍스트.
출력: Redflag 태깅 + 5층 진단 + 리스크 매트릭스 + 변호사 상담 필요도.

① 계약 기초정보 추출(당사자·유형·준거법·기간)
② Redflag 12+K10 스크린 → Red/Yellow/Green 태깅
③ L1 의무·권리 매트릭스
④ L2 리스크 3×3
⑤ L3 Playbook (핵심 쟁점만 3-Tier)
⑥ L4 Exit 점검
⑦ L5 Korea 오버레이
⑧ 종합 평가·변호사 상담 필요도
