# Redflag Catalog — 12종 글로벌 표준

계약검토 1차 스크린. Red/Yellow/Green 태깅. 출처: LegalOn·Harvey·Spellbook·Sirion·Ironclad 공개자료, Harvard PON, Bellrock Advisory.

## 태깅 규칙

- **Red**: 원칙 거부. Tier3 마지노선 — 삭제 or Walk-away
- **Yellow**: 협상. Tier2 — Cap·Carve-out·기간 제한으로 조정
- **Green**: 통과. 표준 문구

## 12종

### 1. Uncapped Liability / 무한책임 (Red)
**정의:** 배상책임에 금액 상한 없음. 계약금액 초과 무한 배상 가능.
**예문:** "Party A shall be liable for all damages, direct or indirect, arising from breach."
**대응:**
- Tier1: Cap = 계약금액 1x (12개월 fees)
- Tier2: Cap = 2x, IP·기밀 위반은 별도 Cap
- Tier3: Uncapped 수용 시 반드시 보험 부보 확인

### 2. Uncapped Indemnity / 무한보상 (Red)
**정의:** 제3자 클레임에 대한 면책·방어 의무에 상한 없음.
**예문:** "indemnify, defend, and hold harmless ... from any and all claims"
**대응:**
- Tier1: Cap = 계약금액, 제외항목(상대방 과실·불가항력·일반 손해) 카브아웃
- Tier2: IP Indemnity만 Uncapped, 나머지 Cap
- Tier3: 보험 의무 부과

### 3. Liquidated Damages Penalty / 과도위약금 (Yellow)
**정의:** 위약금이 실제 손해 대비 과다. 한국 민법 398조 법원 직권감액 대상.
**대응:**
- Tier1: 실손해 기준 (총계약금의 10% 내)
- Tier2: 단계별 위약금 (경미/중대 차등)
- Tier3: 상한선 명시

### 4. Unilateral Termination / 편측해지 (Red)
**정의:** 일방만 자유해지권 보유. 상대방은 사유 필요.
**대응:**
- Tier1: 양방 동등한 해지권 or 양방 사유 제한
- Tier2: 편측해지 허용하되 Notice 90일+ Termination Fee
- Tier3: 해지사유 구체 열거

### 5. MFN (Most-Favored-Nation) (Yellow)
**정의:** "우리한테 더 나쁜 조건 주면 안 됨" — 소급 적용으로 미래 자동 연동.
**대응:**
- Tier1: MFN 삭제
- Tier2: 범위 제한 (동일 업종·동일 규모·동일 기간)
- Tier3: 이사회 동의 등 절차 장벽

### 6. Change of Control Consent / 지배권 변동 승인 (Yellow)
**정의:** M&A·투자 시 상대방 동의 필요. Deal Killer.
**대응:**
- Tier1: Consent 불필요, Notice only
- Tier2: Consent 필요하되 "not unreasonably withheld"
- Tier3: 경쟁사 대상 제외만 명시

### 7. IP Broad Assignment / IP 포괄양도 (Red)
**정의:** 개발 IP 전부 + 미래·배경 IP까지 포괄 양도.
**대응:**
- Tier1: 계약 대상 과업 IP만 양도. 배경 IP 라이선스로
- Tier2: Work Product만, 배경 IP 유보
- Tier3: 한국 직무발명(특허법 39조) 근로자 이익배려 의무 명시

### 8. Exclusivity / 독점 (Yellow)
**정의:** 특정 지역·시장·고객군 독점 부여. 시장접근 제약.
**대응:**
- Tier1: 독점 ✗, 우선협상권만
- Tier2: 독점 인정하되 실적 연동 해제 조건
- Tier3: 기간·지역·제품 3축 모두 제한

### 9. Arbitration Seat / 준거법 편측 (Yellow)
**정의:** 상대방 본국·유리 법역으로 강제.
**대응:**
- Tier1: 중립 제3지 (싱가포르 SIAC·홍콩 HKIAC) + 중립법
- Tier2: 피고 본국 원칙
- Tier3: 한국계약은 대한상사중재원(KCAB) 우선, 중재지 서울

### 10. Perpetual Confidentiality / 무기한 비밀유지 (Yellow)
**정의:** 계약 종료 후에도 영구 비밀유지.
**대응:**
- Tier1: 기간 제한 (3-5년, 영업비밀은 공개시까지)
- Tier2: 정보 유형별 차등 (일반 3년, 기술 5년, 영업비밀 영구)
- Tier3: 일반조항은 기한제, 특정 Trade Secret만 무기한

### 11. Unilateral Amendment / 일방적 변경권 (Red)
**정의:** 한쪽이 Notice만으로 계약 조건 변경. 약관법 불공정조항.
**대응:**
- Tier1: 양방 서면합의 필수
- Tier2: 변경 시 30일 전 Notice + 상대방 해지권
- Tier3: 법령 변경 대응 조항만 예외 허용

### 12. Non-Compete 과다 / 비경쟁 (Yellow)
**정의:** 비경쟁 기간 장기·지역 광범위·보상 없음.
**대응:**
- Tier1: 근로자 6-12개월, 거래 1-2년. 지역·업무 구체
- Tier2: 보상(퇴직금 상당) 지급 조건
- Tier3: 한국법상 영업비밀 보호 최소범위만, 과도 시 무효

## Green Zone 기준
- 업계 표준 문구 (NDA 기본 비밀유지, 상호 indemnity cap, 표준 Force Majeure)
- 한국 표준계약서 조항 (공정위·문체부·고용부 고시) 원형

## 스크린 수행법
1. 계약서 전문 스캔 → 12종 키워드 매칭 (indemnify, liability, terminate, MFN, change of control, assign, exclusive, arbitration, confidential, amend, non-compete)
2. 각 히트 조항 → Red/Yellow/Green 태깅
3. Red 조항 수 3개+ = 전면 재협상 권고
